## 🎯 项目理念

**Vibe Coding**: 用自然语言描述想法，让 AI 完成代码实现

## 📦 当前项目

### [Calculator App](./calculator-app)
一个基于 Java Swing 的图形计算器应用。
- **技术**: Java / Swing
- **功能**: 基础四则运算、表达式解析

### [Todo App](./todo-app)
待办事项管理工具。
- **技术**: HTML / CSS / JavaScript
- **功能**: CRUD 操作与状态管理

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/your-repo/web-demo.git

# 进入项目目录
cd web-demo

# Calculator App (Java)
cd calculator-app
javac -d out src/main/java/com/calculator/*.java
java -cp out com.calculator.Calculator

# Todo App (Web)
cd todo-app
# 直接用浏览器打开 index.html
```

## 🔄 Vibe Coding 工作流

1. **提出想法** → "给我做一个计算器"

3. **验证运行** → javac & java 编译运行
4. **自动推送** → git add & commit & push
