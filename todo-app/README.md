# 待办事项 (Todo App)

一个简洁美观的待办事项管理应用，使用原生 HTML、CSS 和 JavaScript 构建。

## 项目简介

这是一个轻量级的待办事项应用，无需任何框架或构建工具，直接在浏览器中运行。支持添加、删除、标记完成待办事项，数据自动保存到本地存储。

## 已完成功能

- ✅ 添加新的待办事项
- ✅ 删除待办事项
- ✅ 标记/取消标记完成状态
- ✅ 三种过滤视图（全部/进行中/已完成）
- ✅ 清除已完成事项
- ✅ 本地存储持久化
- ✅ 响应式设计，支持移动端

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 在输入框中输入待办事项，按回车或点击"添加"按钮
3. 点击圆框标记事项完成
4. 使用顶部标签切换不同视图
5. 点击"清除已完成"删除所有已完成事项

## 技术栈

- HTML5
- CSS3 (渐变、Flexbox、动画)
- 原生 JavaScript (ES6+)
- LocalStorage API
